Implemented 8 frame management opcode handlers: call/ret/enter/leave for procedure frames, loadl/storel for local variable access, loada/storea for argument access. Args stay on eval stack with saved_esp pointer for access. ret auto-detects return values by comparing esp to saved_esp. Tested with recursive factorial(5)=120 printed as 'x'. Added ret_temps static storage for the complex ret handler, and proper call/eval stack segment spacing.
