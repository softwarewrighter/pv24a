Implemented jmp/jz/jnz control flow opcode handlers. op_jmp sets pc to addr24 unconditionally. op_jz/op_jnz pop flag from eval stack and conditionally set pc. All three read 3-byte absolute address operands from the code stream. Wired handlers into dispatch table at 0x30-0x32. Test bytecode runs countdown loop from 5→0, printing '54321\n' via PUTC. Verified correct output with cor24-run.
