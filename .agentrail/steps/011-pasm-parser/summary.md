Implemented two-pass p-code assembler parser in pasm.s. Reads all UART input into buffer, pass 1 collects symbols (labels, procs, consts, globals, data) and computes sizes, pass 2 emits bytecode with resolved references. Includes mnemonic table for all 43 opcodes, symbol table with name pool, emit helpers, and bytecode dump. Verified with 5 reg-rs tests: push_halt, push_s_sys, const_directive, label_jmp, data_ref. All produce correct bytecode.
